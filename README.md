# Typeframe

Premium **text-to-image** studio. Turn words into shareable images with editorial typography, themes, and instant canvas preview.

## Stack

- [Astro](https://astro.build) 6
- TypeScript
- Vanilla CSS
- HTML Canvas API + SVG export

## Commands

```bash
npm install
npm run dev      # http://localhost:4321
npm run build
npm run preview
```

## Features

- Plain / rich / markdown text modes with live preview
- 9 templates (quote, social, study notes, code snippet, …)
- 8 themes (Midnight, Paper, Glass, Editorial, …)
- Full typography controls
- Backgrounds: solid, gradient, mesh, glass, noise, upload
- Export: PNG, JPEG, WebP, SVG at preset or custom resolution
- Drag & resize text blocks with guides
